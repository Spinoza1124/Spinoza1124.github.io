vim快捷键

如何使用vimv

在终端中输入vim命令，进入vim编辑器。vim有三种模式：普通模式、插入模式和命令模式。

  • 普通模式：默认模式，可以使用各种快捷键进行文本编辑。
  • 插入模式：用于输入文本，可以使用i、a等命令进入。
  • 命令模式：用于执行命令，可以使用:进入。
    在普通模式下，按i进入插入模式，按Esc返回普通模式。
    在插入模式下，按Esc返回普通模式，按:进入命令模式。
    在命令模式下，可以输入各种命令，如保存、退出等。

命令模式

在命令模式下，可以输入各种命令，如保存、退出等。常用命令如下：

  • :w：保存文件
  • :q：退出vim
  • :wq：保存并退出vim
  • :q!：强制退出vim，不保存文件
  • :e filename：打开文件
  • :help：查看帮助文档

普通模式

在普通模式下，可以使用各种快捷键进行文本编辑。常用快捷键如下：

  • h：向左移动光标
  • j：向下移动光标
  • k：向上移动光标
  • l：向右移动光标
  • 0：移动到行首
  • $：移动到行尾
  • gg：移动到文件开头
  • G：移动到文件结尾
  • x：删除光标所在字符
  • dd：删除光标所在行
  • yy：复制光标所在行
  • P(大写)：粘贴剪切板内容到光标上方
  • p(小写)：粘贴剪切板内容到光标下方
  • u：撤销上一次操作
  • v：进入可视模式，可以选择文本
  • y：复制选中的文本
  • d：删除选中的文本
  • c：剪切选中的文本
  • r：替换光标所在字符
  • .：重复上一个操作
  • Ctrl + r：重做上一个操作
  • :w:保存文件
  • :q:退出vim
  • :wq:保存并退出vim
  • :q!:强制退出vim，不保存文件

插入模式

在插入模式下，可以输入文本。常用快捷键如下：

  • i：在光标前插入文本
  • I：在行首插入文本
  • a：在光标后插入文本
  • A：在行尾插入文本
  • o：在光标下方插入新行
  • O：在光标上方插入新行
  • Esc：返回普通模式

可视模式

在可视模式下，可以选择文本。常用快捷键如下：

  • v：进入可视模式
  • V：进入行可视模式
  • Ctrl + v：进入块可视模式
  • y：复制选中的文本
  • d：删除选中的文本
  • c：剪切选中的文本
  • p：粘贴
  • u：撤销

vim按键说明

第一部分：一般模式的光标移动，复制粘贴、搜索替换等

移动光标的方法

操作 说明
ctrl + f 屏幕向下移动一页
ctrl + b 屏幕向上移动一页
ctrl + d 屏幕向下移动半页
ctrl + u 屏幕向上移动半页
n<space> 按下数字后再按空格键，光标会向右移动这一行的n个字符。例如20则光标会向后面移动20个字符距离
0 光标移动到行首
$ 光标移动到行尾
gg 光标移动到文件开头
G 光标移动到文件结尾
n<Enter> 光标移动到当前行的第n个字符
nG 光标移动到第n行

搜索替换

操作 说明
/word 向光标之下寻找一个名称为word的字符串。例如要在档案内搜寻vbird这个字符串，就输入/vbird即可
?word 向光标之上寻找一个名称为word的字符串
n 搜寻下一个
N 搜寻上一个
:%s/old/new/g 将档案内的old字串替换成new字串

复制、剪切、粘贴

操作 说明
x 删除光标所在的字符
X 删除光标前的字符
dd 删除光标所在的行
ndd n为数字。剪切光标所在的向下n行,例如20dd则是剪切20行，用p/P可以粘贴
d1G 删除光标所在行到第一行的所有数据
dG 删除光标所在行到最后一行的所有数据
d0 那个是数字的0,删除游标所在处，到该行的最前面的一个字符
yy 复制光标所在行
nyy n为数字。复制光标所在行向下n行,例如20yy则是复制20行，用p/P可以粘贴
y1G 复制光标所在行到第一行的所有数据
yG 复制光标所在行到最后一行的所有数据
y0 复制光标所在行到该行的最前面的一个字符
y$ 复制光标所在的那个字符到该行行尾的所有数据
p 粘贴到光标后面
P 粘贴到光标前面
J 将光标所在行与下一行合并
u 撤销上一个操作
c 重复删除多个数据，例如向下删除10行, [10cj]
